📁 Local File

Chrome / Edge only

Save your data to a .json file anywhere on your computer — Documents, Dropbox folder, USB drive, whatever you like. Changes auto-save every time you make them. No account needed.

✅ Connected to local file — auto-saving

🟢 Google Drive

Requires setup

Sync data to a file called storewise-data.json in your personal Google Drive. Each user connects their own Google account — you never see their data.

✅ Connected to Google Drive — auto-saving
⚙️ Setup instructions
  1. Go to console.cloud.google.com
  2. Create a project → Enable Google Drive API
  3. OAuth consent screen → External → Add your GitHub Pages URL as authorized domain
  4. Credentials → Create OAuth Client ID → Web application
  5. Add https://stiliilist.github.io/storewise-inventory-app/ as Authorized redirect URI
  6. Copy the Client ID and paste it in app.js at the top: GOOGLE_CLIENT_ID: 'your-id-here'

🔵 Dropbox

Requires setup

Sync data to /storewise-data.json in each user's personal Dropbox. Works across devices automatically once connected.

✅ Connected to Dropbox — auto-saving
⚙️ Setup instructions
  1. Go to dropbox.com/developers/apps
  2. Create App → Scoped access → Full Dropbox
  3. Under OAuth 2, add your GitHub Pages URL as Redirect URI:
    https://stiliilist.github.io/storewise-inventory-app/
  4. Copy the App key and paste it in app.js: DROPBOX_APP_KEY: 'your-key-here'
