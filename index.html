🔐 PIN Protection

Your app is protected by a PIN. All data is encrypted with AES-256-GCM using your PIN as the key — even if someone opens the browser storage, they see only encrypted gibberish without your PIN.

Auto-lock

The app automatically locks after 15 minutes of inactivity. This is set in CONFIG.AUTO_LOCK_MS in app.js. Set to 0 to disable.

✅ All data in browser storage (localStorage)

✅ Files saved to local disk via File System API

🟢 Google Drive

Requires setup

Saves an encrypted file (storewise-data.enc) to each user's personal Google Drive. Google cannot read the contents.

✅ Connected to Google Drive — auto-saving encrypted
⚙️ Setup instructions
  1. Go to console.cloud.google.com
  2. Create project → Enable Google Drive API
  3. OAuth consent → External → add your GitHub Pages domain
  4. Credentials → OAuth Client ID → Web app
  5. Add your GitHub Pages URL as Redirect URI
  6. Paste Client ID into CONFIG.GOOGLE_CLIENT_ID in app.js

🔵 Dropbox

Requires setup

Saves an encrypted file to each user's Dropbox. Dropbox cannot read the contents. Works across all devices.

✅ Connected to Dropbox — auto-saving encrypted
⚙️ Setup instructions
  1. Go to dropbox.com/developers/apps
  2. Create App → Scoped → Full Dropbox
  3. Add your GitHub Pages URL as Redirect URI
  4. Paste App Key into CONFIG.DROPBOX_APP_KEY in app.js
